# 🤖 AI Coding Mentor

An AI-powered coding mentor built with Python and the Gemini API.

Unlike a traditional coding judge, this project not only evaluates code but also provides intelligent feedback, explains the logic, analyzes complexity, and suggests better approaches to help users improve their programming skills.

---

## ✨ Features

- 📄 Reads programming questions from text files
- 💻 Accepts user Python solutions
- 🧠 Uses Google's Gemini API for AI-powered code review
- 📊 Explains:
  - Correctness
  - Logic
  - Time Complexity
  - Space Complexity
  - Coding Mistakes
  - Better Approaches
  - Alternative Solutions
  - Pythonic Improvements
- 🏁 Gives a final verdict

---
